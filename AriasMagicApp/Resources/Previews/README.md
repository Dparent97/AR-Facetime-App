## Technical Specifications

### Image Format
- **Format:** PNG with transparency
- **Resolution:** 512x512 pixels
- **Color Space:** sRGB
- **Bit Depth:** 8-bit RGBA

### Visual Requirements
- **Background:** Transparent
- **View Angle:** 3/4 view (slight angle, not straight-on)
- **Lighting:** Well-lit, soft shadows
- **Expression:** Happy/neutral, friendly
- **Framing:** Consistent across all characters (head to mid-torso or full body)

### Style
- Matches 3D character appearance
- Clear and recognizable
- Child-friendly presentation
- Professional quality

## How to Create

### Option 1: Render from 3D Software
1. Open character model in Blender or Reality Composer Pro
2. Set up 3/4 view camera angle
3. Apply good lighting (3-point lighting recommended)
4. Render at 512x512 or higher, then resize
5. Remove background (make transparent)

### Option 2: Screenshot from AR App
1. Place character in well-lit AR environment
2. Capture screenshot at optimal angle
3. Crop and resize to 512x512
4. Use image editor to remove background
5. Export as PNG with transparency

### Option 3: 2D Illustration
1. Create matching 2D artwork
2. Ensure style consistency with 3D models
3. Export at 512x512 with transparent background
